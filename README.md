# PHP 7.2 + Maria DB

Hambiente para desenvovliemtno com PHP 7.2 e Maria DB

Esse container usa o Bind Volume do Docker para manter os arquivos.

### Instalação!
```sh
$ docker-compose up
```
OBS.: O nome do seu container será o nome da sua pasta, para personalizar use:
```yml
  container_name: 
     my-db-container
```

### Acessando host!
```sh
127.0.0.1:8888
```
ou 
```sh
lcoalhost:8888
```
